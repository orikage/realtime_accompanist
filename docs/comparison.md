# Similar Products and Positioning

## Microsoft MySong / Songsmith

Closest conceptually.
It takes a vocal melody and generates matching chords/accompaniment.
However, it is older and closer to recorded composition assistance than a live MIDI instrument.

Positioning against it:

- Songsmith: sing/record melody, generate accompaniment
- This project: play MIDI melody live, show key/chord uncertainty, accompany in realtime

## DigiTech TRIO / TRIO+

A guitar pedal that listens to guitar playing and generates bass/drums.
Very relevant as a product analogy.

Positioning:

- TRIO: guitar chord/rhythm teaching → bass/drums
- This project: keyboard melody-only input → key/chord candidates → full accompaniment

## CASIO Chordana Composer / Trackformer

Short melody motif or input phrase can generate a track.
Closer to auto-composition than live accompaniment.

Positioning:

- Chordana: motif → song generation
- This project: ongoing melody → realtime adaptive accompaniment

## Humtap

Humming and tapping to generate music.
Smartphone AI creation tool.

Positioning:

- Humtap: casual song creation from voice/tap
- This project: live playable instrument for Maker Faire

## Google Magenta A.I. Duet

User plays notes; AI responds with piano phrases.
It is close in terms of real-time human-AI musical interaction, but it is call-and-response rather than accompaniment.

Positioning:

- A.I. Duet: AI replies to your phrase
- This project: AI accompanies your phrase

## Arranger Keyboards

Yamaha/KORG arranger keyboards generate accompaniment based on detected chords.
Usually require left-hand chord input or explicit chord detection area.

Positioning:

- Arranger keyboard: chord input → accompaniment
- This project: melody input → inferred chord candidates → accompaniment

## One-line Differentiation

Existing auto-accompaniment tools usually need chords, recording, or guitar strumming.
This project focuses on realtime right-hand melody interpretation with visible uncertainty and adaptive accompaniment.
